📖 Quick Guide

How to Use

  1. Ask questions in natural language
  2. The system converts your question to a SPARQL query
  3. Results are retrieved from the knowledge graph
  4. Results are displayed in an easy-to-read format

Example Queries

  • Character Info: "Who is Luke Skywalker?"
  • Counting: "How many planets are there?"
  • Filtering: "List all droids"
  • Properties: "What species is Yoda?"
  • Relationships: "Who is Han Solo married to?"
